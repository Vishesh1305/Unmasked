UNMASKED DISCHARGED

"Is this the right moment to wear this mask?"

You awaken in an impossibly vast maze of repeating corridors. Three ancient masks rest before you—each granting a single-use power, each consuming a piece of your identity forever. Escape is possible, but every choice is irreversible.

View on GitHub

This is not a hospital.
It is a containment.

The corridors are endless because the mind is looping. The maze exists to test those who refuse to let go—refused treatment, refused to forget, refused to accept loss.

The architecture defies logic: halls stretch into darkness, doors lead to rooms already visited, walls shift when unobserved. This is not a place built by human hands.

The system did not heal you. It contained you.

The Three Masks

Each mask is a coping mechanism. Each use is a piece of yourself surrendered permanently.

SIGHT

The Hollow Mask

"If I just knew the way out..."

Grants TRUE SIGHT for 15 seconds. Reveals hidden paths, correct doors, and invisible clues.

  • World shifts to high-contrast monochrome
  • Correct paths glow with ethereal light
  • Whispered voices grow desperate as time runs out
SINGLE USE • IRREVERSIBLE
FORM

The Fractured Mask

"If I don't face it directly..."

Allows PHASE through one sealed door, wall, or barrier for 15 seconds.

  • Hands become translucent
  • Ghostly double-image effect
  • Reality tears with crackling static
SINGLE USE • IRREVERSIBLE
SILENCE

The Wailing Mask

"If I make no sound..."

Creates absolute SILENCE. Sound triggers no reactions. The maze cannot hear you.

  • Scream cuts to absolute void
  • Only heartbeat remains, growing louder
  • Sound floods back disorientingly
SINGLE USE • IRREVERSIBLE

Warning: Using a mask at the wrong moment traps you forever. The maze remembers.

Ambient Telemetry

Irreversible Choice

Every mask worn is permanently lost. The game features three distinct acts, each with increasing psychological tension.

01

The Threshold

Clean corridors that degrade. Temptation to use masks early.

02

The Deep Maze

Two mandatory mask uses required. Poor choices result in permanent softlock.

03

The Escape

Final gauntlet. The maze fights back. One mask remains mandatory.

Face the Truth

The masks remember their creators. Will you wear the wrong face?
